# `capuchin_crypt` Security

## LLM-Generated Security Report Policy

Absolutely no security reports will be accepted that have been generated by LLM
agents.

## Supported Versions

Security reports are accepted for the most recent major release with support for
the previous major version ending immediately.

`capuchin_crypt` supports the two most recent major versions of Gleam
(currently 1.14 and 1.15)
and is tested on Node 24, the latest Bun, the latest Deno, and Erlang OTP 27
and 28.

## Reporting a Vulnerability

Create a [private vulnerability report][advisory] with GitHub.

[advisory]: https://github.com/halostatue/capuchin_crypt/security/advisories/new
